# 📜 Code of Conduct

## Our Pledge

We, as contributors and maintainers, are committed to making participation in the API Shield community a welcoming, respectful, and harassment-free experience for everyone, regardless of age, experience level, nationality, ethnicity, gender identity, sexual orientation, disability, or religion.

Our goal is to build an open-source community where developers can collaborate, learn, and grow together.

---

# Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and inclusive.
- Providing constructive feedback.
- Accepting different viewpoints and experiences.
- Showing empathy towards other community members.
- Helping new contributors.
- Focusing on what is best for the project and community.

---

# Unacceptable Behavior

The following behavior will not be tolerated:

- Harassment or abusive language.
- Personal attacks or insults.
- Hate speech or discrimination.
- Trolling or deliberately disruptive behavior.
- Publishing private information without permission.
- Any conduct that could make others feel unsafe or unwelcome.

---

# Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Maintainers may remove, edit, or reject comments, commits, issues, pull requests, or other contributions that violate these guidelines.

---

# Scope

This Code of Conduct applies to all project spaces, including:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation
- Community conversations
- Any official communication channels related to API Shield

---

# Reporting Issues

If you experience or witness unacceptable behavior, please report it by opening a confidential issue or contacting the project maintainer.

All reports will be reviewed fairly and handled with respect for everyone's privacy.

---

# Enforcement

Contributors who violate this Code of Conduct may receive:

1. A warning.
2. Temporary suspension from the community.
3. Permanent removal from the project for repeated or severe violations.

The enforcement decision will depend on the severity and frequency of the violation.
